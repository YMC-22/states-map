== Description ==

The interactive map will display information for each US state in a pop-up window.

== Installation ==

1. Activate Plugin or upload the entire 'ymc-states-map' folder to the '/wp-content/plugins/' directory.
2. Add new States Map US
3. Copy States Map US shortcode and paste to any page or post
4. Set setting for each post
